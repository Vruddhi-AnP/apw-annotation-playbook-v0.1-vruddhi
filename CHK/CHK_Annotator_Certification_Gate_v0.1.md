Annotator Certification Gate Checklist

Inputs Ready:
- [ ] Annotator has access to the annotation tool
- [ ] Label specification is reviewed
- [ ] Edge cases are understood

Process OK:
- [ ] Training quiz is completed
- [ ] Passing score is achieved
- [ ] Questions or doubts are clarified

Outputs OK:
- [ ] Annotator certified to start work
- [ ] Certification status recorded
- [ ] Batch can be assigned

Stop-the-line Checks:
- [ ] STOP if training quiz is not passed
- [ ] STOP if label rules are unclear
- [ ] STOP if tool access is not working
